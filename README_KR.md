# 🎮 Discord RPG Bot v3

## 📋 기능 목록

### 🗺️ 탐험/월드
- 버튼식 이동 (↑↓←→)
- 바이옴 시스템 (평원/숲/사막/설원/화산/바다/동굴)
- 미니맵 오버뷰 (전체 월드 축소맵)
- 랜드마크: 마을(100,100), 던전(50,50), 콜로세움(150,150), 상점가, 낚시터, 이벤트존
- NPC 상호작용 (퀘스트 마스터, 상인, 대장장이, 길드 관리인, PVP 관리인)
- 집 상호작용 (구매/입장/휴식)
- 귀환 스크롤 (마을 순간이동)

### ⚔️ 전투/스킬
- 일반 사냥 (10종 몬스터)
- 던전 시스템 (5층 + 보스)
- 스킬 시스템 (전사/궁수/마법사/성직자/도적 각 4개)
- 협동 레이드 보스전 (HP 공유, 데미지 비례 보상)
- PVP 시스템 (콜로세움 전용)

### 🎒 아이템/장비
- 600종+ 아이템 (무기/방어구/소비/재료/도구)
- 강화 시스템 (+1~+15, 확률 기반)
- 제작 시스템 (6종 특수 아이템)
- 장비 착용 시스템

### 🏪 거래소
- 매물 등록 (버튼 방식)
- 입찰/낙찰 시스템
- 거래소 알림 설정 (키워드 DM 알림)
- 자동 만료 처리 (24시간)

### 🎮 미니게임
- 낚시 (타이밍 리액션 방식)
- 슬롯머신 (일일 10회 제한)
- 주사위 도박
- 낚시 대회 (주간 순위)
- 퀴즈/OX 이벤트

### 👥 소셜
- 결혼 시스템
- 의형제 시스템
- 길드 창설/가입
- 초대 보상 시스템
- 감정표현 이모트
- 음성 채널 연동 (경험치 보너스)

### 📜 퀘스트/업적
- NPC 퀘스트 (5종)
- 업적 시스템 (8종)
- 칭호 시스템

### 🎫 배틀패스
- 무료/프리미엄 트랙
- 시즌 리셋 및 보상

### 🛠️ 개발자 명령어
| 명령어 | 설명 |
|--------|------|
| `/dev_give_item` | 아이템 지급 |
| `/dev_give_coin` | 코인 지급 |
| `/dev_give_gem` | 젬 지급 |
| `/dev_item_list` | 아이템 목록 조회 |
| `/dev_admin_report` | 관리자 리포트 DM |
| `/dev_reset_ranking` | 랭킹 초기화 |
| `/dev_season_reset` | 시즌 리셋 |

## 🚀 실행 방법

1. `.env.example`을 `.env`로 복사
2. `.env`에 봇 토큰 및 개발자 ID 입력
3. 패키지 설치: `pip install -r requirements.txt`
4. 실행: `python main.py`

## 📌 주요 슬래시 명령어

| 명령어 | 설명 |
|--------|------|
| `/rpg` | 게임 시작 (버튼 UI) |
| `/랭킹` | 글로벌/서버 랭킹 |
| `/레이드생성` | 협동 레이드 보스 생성 |
| `/결혼제안 @유저` | 결혼 제안 |
| `/의형제 @유저` | 의형제 맺기 |
| `/감정표현 [동작]` | 이모트 출력 |
| `/초대` | 초대 코드 생성 |
| `/초대등록 [코드]` | 초대 코드 등록 |
| `/버그제보 [내용]` | 버그 제보 |
| `/퀴즈 [문제] [정답]` | OX 퀴즈 이벤트 (관리자) |
| `/공지 [내용]` | 전체 공지 (개발자) |
